# Módulo Jóvenes / Adolescentes (12-17 años)

> Rellena este módulo si en tu núcleo hay adolescentes.

| # | Pregunta | Tu respuesta |
|---|----------|--------------|
| E1 | ¿Cuál es su nivel de autonomía? (puede estar solo en casa, se mueve solo por el barrio, sale solo por la ciudad) | |
| E2 | ¿Tiene teléfono móvil propio? (sí/no, número) | |
| E3 | ¿Tiene conocimientos útiles? (primeros auxilios, scout, orientación, mecánica básica, informática) | |
| E4 | ¿Cuáles son sus horarios habituales fuera de casa? (instituto, extraescolares, amigos) | |
| E5 | ¿Tiene carnet de conducir? (ciclomotor, coche) | |
| E6 | ¿Qué rol podría asumir en emergencia? (mensajero entre núcleos, cuidar hermanos pequeños, ayudar a abuelos, tareas físicas) | |
| E7 | Centro de estudios: nombre, dirección, teléfono | |
