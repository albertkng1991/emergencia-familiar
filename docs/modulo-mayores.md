# Módulo Personas Mayores / Dependientes

> Rellena este módulo si en tu núcleo hay personas mayores con necesidades especiales o algún grado de dependencia.

| # | Pregunta | Tu respuesta |
|---|----------|--------------|
| D1 | ¿Cuál es su nivel de movilidad? (autónomo, usa bastón, andador, silla de ruedas, encamado) | |
| D2 | Lista DETALLADA de medicación: nombre del medicamento, dosis, frecuencia, y si es esencial (sin ella hay riesgo vital) | |
| D3 | ¿Usa aparatos eléctricos médicos? (CPAP, concentrador de oxígeno, nebulizador, etc.) Para cada uno: ¿tiene batería? ¿Cuánta autonomía? ¿Hay alternativa sin electricidad? | |
| D4 | ¿Tiene deterioro cognitivo? (ninguno, leve, moderado, Alzheimer, demencia) | |
| D5 | ¿Puede evacuar a pie? ¿Cuántos metros puede caminar? ¿Puede bajar escaleras? ¿Necesita ayuda? | |
| D6 | ¿Quién es su cuidador principal? (nombre) | |
| D7 | Centro de salud habitual: nombre, teléfono, dirección | |
| D8 | ¿Tiene grado de dependencia reconocido? ¿Cuál? | |
