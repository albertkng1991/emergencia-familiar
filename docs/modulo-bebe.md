# Módulo Bebé / Niños Pequeños (0-5 años)

> Rellena este módulo si en tu núcleo hay bebés o niños menores de 6 años.

| # | Pregunta | Tu respuesta |
|---|----------|--------------|
| C1 | ¿Cómo se alimenta el bebé? (lactancia materna, fórmula — marca y tipo —, mixta, ya come sólidos) | |
| C2 | ¿Qué talla de pañales usa? ¿Marca habitual? ¿Cuántos tienes de reserva? | |
| C3 | ¿Tienes portabebés? ¿De qué tipo? (mochila ergonómica, fular, bandolera) | |
| C4 | ¿Qué grupo de silla de coche usa? ¿En qué vehículo está instalada? | |
| C5 | ¿Va a guardería? Nombre, dirección y teléfono de la guardería | |
| C6 | Si ambos padres no estuvieran disponibles, ¿quién puede hacerse cargo? (nombre, teléfono, relación con el niño) — indica al menos 2 personas | |
| C7 | ¿Qué medicación infantil tienes en casa? (Apiretal, Dalsy, suero oral, etc.) | |
| C8 | Nombre y teléfono del pediatra | |
| C9 | ¿Hay alimentos específicos imprescindibles? (marca de fórmula concreta, cereales sin gluten, etc.) | |
| C10 | ¿Cómo es su rutina de sueño? (útil para cuidadores alternativos) | |
