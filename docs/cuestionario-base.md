# Cuestionario Base — Plan de Emergencia Familiar

> Rellena este cuestionario con los datos de tu núcleo familiar.
> Todos los campos son opcionales excepto los marcados con (*).
> Cuanta más información aportes, más personalizado será tu plan.

---

## BLOQUE 1: Miembros del núcleo

Para CADA persona que vive habitualmente en tu hogar:

| Campo | Descripción |
|-------|-------------|
| Nombre (*) | Nombre completo |
| Edad (*) | Años |
| DNI | Número de documento |
| Grupo sanguíneo | A+, B-, O+, AB+, etc. |
| Teléfono | Número de móvil |
| Alergias | Alergias conocidas (medicamentos, alimentos, picaduras) |
| Medicación fija | Medicamentos que toma a diario |
| Condiciones médicas | Enfermedades crónicas, discapacidades, etc. |
| Parentesco (*) | titular, cónyuge, hijo, padre, etc. |

---

## BLOQUE 2: Vivienda

| Campo | Descripción |
|-------|-------------|
| Dirección (*) | Dirección completa |
| Tipo (*) | Piso, adosado, chalet, etc. |
| Planta | Si es piso, ¿en qué planta? |
| Ascensor | ¿Tiene ascensor? (sí/no) |
| Garaje | ¿Tiene garaje? ¿Dónde? |
| Trastero | ¿Tiene trastero? ¿Dónde? |
| Llave de paso del agua | ¿Dónde está? |
| Llave de paso electricidad | ¿Dónde está el cuadro eléctrico? |
| Llave de paso del gas | ¿Dónde está? |
| Espacio exterior | ¿Tienes balcón, terraza, jardín? |
| Habitación segura | ¿Hay una habitación interior sin ventanas? ¿Cuál? |

---

## BLOQUE 3: Transporte

Para CADA vehículo:

| Campo | Descripción |
|-------|-------------|
| Tipo | Coche, moto, furgoneta, bicicleta |
| Modelo | Marca y modelo |
| Plazas | Número de plazas |
| Ubicación | ¿Dónde se aparca habitualmente? |
| Todoterreno | ¿Es todoterreno/4x4? |

- **¿Quién tiene carnet de conducir?** (nombres)
- **¿Cuántas bicicletas hay?**

## BLOQUE 4: Recursos y capacidades

| Campo | Descripción |
|-------|-------------|
| Conocimientos útiles | ¿Alguien del núcleo es médico, electricista, mecánico, tiene formación en primeros auxilios...? |
| Herramientas | ¿Qué herramientas tenéis? (caja de herramientas, generador, motosierra...) |
| Armas de caza | ¿Tiene algún miembro licencia de armas/caza? |
| Huerto/animales | ¿Tenéis huerto, gallinas, otro ganado? |
| Propiedad secundaria | ¿Tenéis casa de campo, apartamento en otra ciudad, finca? (tipo, ubicación, descripción) |

## BLOQUE 5: Suministros actuales

| Campo | Descripción |
|-------|-------------|
| Comida no perecedera | ¿Tenéis conservas, arroz, pasta almacenado? ¿Para cuántos días? |

| Copias de documentos | ¿Tenéis copias de los documentos importantes? ¿En USB, nube, casa de familiar? |

## BLOQUE 6: Comunicaciones

| Campo | Descripción |
|-------|-------------|
| Móviles (*) | Para cada móvil: nombre del dueño, número, operador |

| Campo | Descripción |
|-------|-------------|

## BLOQUE 8: Preferencias

| Campo | Descripción |
|-------|-------------|

> **IMPORTANTE**: Una vez rellenes este cuestionario, pásale los datos al coordinador para que los introduzca en el sistema.
> Si tu núcleo tiene **bebés/niños pequeños**, **personas mayores** o **adolescentes**, rellena también el módulo correspondiente.
